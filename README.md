# wechat-dribbble
🍁🌱🌿wechat app for dribbble

> write less, do more 

## 简介
 - 利用宇宙最美的设计网站[dribbble](https://dribbble.com/shots) 提供的API完成功能
 - 响应式布局，各个分辨率均能展现良好

## 使用
 - 根据[awesome-wechat-weapp](awesome-wechat-weapp)提供的方法，安装微信小程序开发环境
 - `git clone https://github.com/nicesu/wechat-dribbble.git`
 - 在微信开发工具中添加刚刚下载的项目项目
